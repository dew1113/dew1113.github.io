发表于 2016-01-30   |   分类于 技术   |  

【1.CSS选择器常见的有几种?】

常见的css选择器有：

  • 全局选择器：例如*{margin:0;padding:0}
  • id选择器：例如#box{margin:0;padding:0}
  • 类选择器：例如.box{margin:0;padding:0}
  • 标签选择器：例如a{text-decoration:none;}
  • 组合选择器：例如 a，p{color:red;}
  • 属性选择器：例如input[type=“text”]{color:blue;}
  • 伪类选择器:例如 a:hover{color:red;}
  • 继承选择器:例如li a{color:red}
  • 子元素选择器：例如 ul > li{border:1px solid red;}
  • !important选择器：例如 p{color:red； !important}

【2.选择器的优先级是怎样的】

从高到低分别是:

  • 在属性后面使用 !important 会覆盖页面内任何位置定义的元素样式
  • 作为style属性写在元素标签上的内联样式
  • id选择器
  • 类选择器
  • 伪类选择器
  • 属性选择器
  • 标签选择器
  • 通配符选择器
  • 浏览器自定义
    每个选择器本身有优先级，越具体优先级越高;

【3.class 和 id 的使用场景?】

  • ID选择器常作用于大范围，标示度高的标签上.如页面布局的头部、内容区域和页脚，因为id在一个文档中只能被使用一次。
  • class更多地被应用于文字板块及页面修饰等方面。

###【4.使用CSS选择器时为什么要划定适当的命名空间？】
为了准确定位到当前元素，只对所选的命名空间有效，避免与其他元素冲突。

【6.列出你知道的伪类选择器】

  • :link 未被点击的连接
  • :visited 已被点击的连接
  • :active 鼠标按下没有释放的元素
  • :hover 鼠标悬停的元素
  • :focus 获得焦点的元素
  • :first-child 第一个子元素
  • :last-child 最后一个子元素
  • :first-of-type 同类型元素父级的第一个子元素
  • :last-of-type 同类型父级的最后一个子元素

【7.first-of-type和first-child的作用和区别？】

  • :first-child匹配的是某父级元素的第一个子元素，可以说是结构上的第一个子元素。例如：p:first-child 必须满足两个条件。第一必须是p元素；第二该p元素还必须是第一个子元素。两者都符合才会被选中。

  • :first-of-type匹配的是某父元素下该类型子元素中的第一个，比如p: first-of-type就是指所有类型为p的子元素中的第一个。这里不再限制是第一个子元素，只要是该类型元素的第一个就行。

【9.text-align: center的作用是什么，作用在什么元素上？能让什么元素水平居中】

text-align: center是文本对齐的一种方式，文本居中对齐。
只能对块级元素内容生效。所谓的块级元素就是指<div>、<p>、<h1>、<ol>、<ul>、<dl>、<table>、<address>、<blockquote>、<form>等等。
作用： 能够让块级元素的文本内容水平居中。
