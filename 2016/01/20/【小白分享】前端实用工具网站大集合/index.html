工具篇

  1. 编辑器哪家强 sublime text3
    啥也不用说 他特别特别轻 轻如鹅毛 打开嗖嗖嗖嗖

  1. sublime 实用插件

    • Emmet 自动补全代码；
    • Autofilename 自动补全路径；
    • Livereload 跟chrome一起使用实时刷新页面；

    • Colorpicker
    • ……
      百度一下 有很多。。。按需选择

  2. 越狱好帮手 chromeGAE
    下载地址
    下载包里应该是有使用说明的，安装完以后打开代理通道那个exe就可以了。

  3. 录屏制作gif———LICEcap
    这个不算前端必要工具，但是我觉得很好用，在自己写博客的时候，或者记录学习心得的时候，可以自己写一段代码，用它录下来，可以很清晰看到实现效果。对以后复习啊，或者跟别人交流都挺好的。

  4. fontawesome和iconfont
    图标库。后者是阿里的，更适合国内使用。而且可以定制，前提是。。你会画出来。。。

  5. git
    现在用到的也不多，但是需要了解一下。掌握几个基本命令，可以用于操作github。
    比如git pull, git add ,git commit,git push最基本的一些。

  6. github
    代码管理。除了托管代码，查看别人的开源代码外，因为它提供有比较大的免费空间和流量，可以在上面搭建自己的个人博客。而且最棒的是可以绑定自己的个人域名。
    搭建hexo+github博客参考教程

  7. markdown
    这也算是一种语言吧。。。大概就十几种写法。

比如

一级标题

三级标题

        width: 300px;
        position: absolute;
        left: 250px;

网站篇

  1. 制作雪碧图 http://csssprites.com/
    雪碧图自己搜索了解一下 ，是个非常好用的东西，对优化网站，提高加载速度比较有用。工作后会常用到。上面链接是制作雪碧图用的在线网站。
    CSS雪碧是将小图标和背景图像合并到一张图片上，然后利用css的background:position来定位图片。
    作用：减少加载网页图片时对服务器的请求次数，减少服务器压力，节约服务器流量，提高页面的加载速度。

  2. Can U use
    这是一个查询css属性兼容性的网站，非常非常实用。拿不定的属性就去查一下，so easy!

  3. 推荐一些比较好的博客

    • 张鑫旭的博客.绝对大牛。我的男神~
    • 廖雪峰的官方网站.里面有非常全面并且专业的git教程，当时git就是从这里学习的。
    • 阮一峰js标准参考教程。这个还未出版，非常好的js教材，很新很贴近实际工作。
    • 大漠的博客.现在在手淘，css届神人。不咋会js,但是把css研究地非常透彻，著有css3权威指南。
