【学习记录】DAY9-CSS常见技巧

发表于 2016-02-02   |   分类于 技术   |  

【1.CSS Sprite(雪碧图)指什么? 有什么作用】

CSS雪碧是将小图标和背景图像合并到一张图片上，然后利用css的background:position来定位图片。
作用：减少加载网页图片时对服务器的请求次数，减少服务器压力，节约服务器流量，提高页面的加载速度。

【2.img标签和CSS背景使用图片在使用场景上有何区别】

如果内容是相对固定的，不可点击的，比如网站logo，一些小icon选用css背景图。
如果图片是可变化可点击的，比如广告图片，则需要用img标签。

【3.title和 alt属性分别有什么作用】

  • title属性可以用在除了base，basefont，head，html，meta，param，script和title之外的所有标签上。title的作用是当鼠标悬停在某一个标签上，会出现对此标签的一个注释。
  • alt只能用于img、area和input。alt是在你的图片因为某种原因不能加载时在页面显示的提示信息（属于替代文字)，它会直接输出在原本加载图片的地方。

【4.background: url(abc.png) 0 0 no-repeat;这句话是什么意思】

将abd.png这个图片设置为背景，位置不偏移，不平铺。

【5.background-size有什么作用? 兼容性如何? 常用的值是?】

background-size:设置背景图片大小。
常用数值：background-size:auto| px px| % %| cover|contain

  • cover：背景图片会将整个背景块覆盖，但不保证图片显示完全
  • contain：按设置的宽高完整显示图片，但不保证背景块全部填充
    设置宽高只有一个值时，按宽度等比缩放。

【7.如何设置元素透明? 兼容性？】

添加:opacity:0.6 (0~1数字)
IE6，IE7浏览器：filter:alpha(opacity=60)(0~100数字)

【8.opacity和rgba都能设置透明，有什么区别】

opacity可以将块里的所有元素已经属性设置为透明；
rgba只能将其中的元素设置透明，并不能改变背景等透明度。
