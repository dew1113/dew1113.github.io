【学习记录】DAY12-三栏布局

发表于 2016-02-06   |   分类于 技术   |  

【1.负边距在让元素产生偏移时和position:relative有什么区别？】

负边距使元素产生偏移时，不会脱离文档流，元素位置依然存在，所以会对后面的元素位置产生影响。
而positon:relative会使得元素脱离文档流，因为对其他元素的位置并不会产生影响。

【2.使用负margin形成三栏布局有什么条件？】

  • 三个div都需要设置左浮动，并且父容器后面要清除浮动；
  • 父容器要设置左右padding为左右边栏预留出空位；
  • 使用position:relative将浮动的左右边栏定位到两侧。

    【3.圣杯布局的原理是怎样的？简述实现圣杯布局的步骤】

    圣杯布局主要就是利用负margin给浮动的元素进行垂直定位，然后再用position定位调整具体位置。
    1. 将三栏块元素设置浮动；
    2. 给左右侧栏添加适合的负边距，产生垂直上的移动；
    3. 给父容器添加左右padding,给侧边栏空余出位置；并且在父容器下清除浮动；
    4. 使用position:relative调整左右侧边栏的位置

【4.双飞翼布局的原理？实现步骤】

双飞翼布局跟圣杯布局类似，只不过不改变父容器的padding。而是在主内容栏里添加一个块，并设为居中自适应宽度。

  1. 在main元素里再添加一个例如wrap名字的div块。
  2. 同上，在main和两个侧边栏上都添加float：left;并且在父容器后面清除浮动；
  3. 设置侧边栏负边距，使得都在同一行里；
  4. 给wrap块增加左右margin,给侧边栏空出位置
