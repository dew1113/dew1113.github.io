【小白分享】如何将hexo+github搭建的博客迁移到另一台电脑上

发表于 2016-07-06   |   分类于 技术   |  

自从放假以后，玩了大半个月，心里觉得特别内疚，决定要好好学习了。照常学习之后会有一个总结的知识点日记，昨天写完日记后突然想起来之前都是在学校机房学习写日记并且部署上去的，回家之后换了电脑，有点懵了，不知道要怎么传了。经过百度加知乎各种方法筛选后，发现了一个特别棒的好方法！

  1. 博客文件放在U盘或者云盘里，如果事前没有存，可以到github上download一下
  2. 换了新电脑后，ssh肯定要重新部署好
  3. 最关键一步，直接把博客文件夹放到新电脑里后，把文件夹下的.deploy_git文件夹删除。然后hexo clean,hexo d两步就搞定了！
